# Food-Exxe-Relo

### Demo - https://larsie08.github.io/react-food-exxe-relo/

## Stack:

1. React
2. React Router
3. React-Hooks
4. Axios
5. PropTypes
6. SCSS

## Installation

```
git clone https://github.com/larsie08/react-food-exxe-relo.git
```

## Installing dependencies

```
npm install
```

## Starting a project

```
npm run server
npm run start
```

## Building a project

```
npm run build
```
